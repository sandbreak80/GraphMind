📊 SYSTEM CAPABILITIES

Research Modes (4):
├─ 📚 Document RAG      → PDFs, videos, documents
├─ 📝 Obsidian          → Personal knowledge vault
├─ 🌐 Web Search        → Real-time SearXNG results
└─ 🔬 Comprehensive     → Combined multi-source

Multi-Format Ingestion:
├─ PDFs (Docling)       → Structure-aware + OCR fallback
├─ Videos (Whisper)     → GPU transcription + timestamps
├─ Excel (openpyxl)     → Data + formulas + comments
├─ Word (python-docx)   → Structure-aware extraction
└─ Text (plain)         → Simple chunking

RAG Pipeline:
├─ BM25 Retrieval       → Keyword-based prefilter
├─ Dense Embeddings     → BAAI/bge-m3 (1024-dim)
├─ Cross-Encoder        → BAAI/bge-reranker-large
├─ GPU Acceleration     → CUDA-optimized
└─ Redis Caching        → Query response caching

Metadata Enrichment:
├─ AI Summaries         → LLM-generated summaries
├─ Key Concepts         → Auto-extracted concepts
├─ Categories           → Document classification
├─ Keywords             → Frequency-based extraction
└─ 20+ Fields           → Rich context per chunk
