🚀 QUICK START

1. Start Ollama:
   ollama pull llama3.1

2. Build & Launch:
   docker-compose build
   docker-compose up -d

3. Ingest All Content (3-4 hours):
   make ingest
   
4. Monitor Progress:
   docker-compose logs -f

5. Query Your Knowledge:
   make test-query
   make extract-spec
